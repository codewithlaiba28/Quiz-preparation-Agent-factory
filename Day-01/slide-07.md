**Ye Pehli Slide Hai - Programming vs Prompting:**

## Paradigm Shift: Code Se Natural Language Tak

**Purana Tareeqa (Left Side - Programming in Python):**
- Developer code likhta tha
- Python/Java/TypeScript seekhna parta tha
- Technical skills zaroori thi
- Syntax, logic, debugging - sab kuch manually

**Naya Tareeqa (Right Side - Prompting in English):**
- Ab aap English mein baat karte ho
- AI samajh kar kaam karta hai
- Natural language hi programming ban gayi
- Non-technical log bhi agents bana sakte hain

## Key Point:
Ye **AI Development** ka naya era hai jahan:
- Code likhne ki jagah English mein instructions dete hain
- Prompting hi nayi programming language hai
- Accessibility barh gayi - har koi agent bana sakta hai

**Ye Three Waves of AI ka hissa hai:**
1. First Wave: Traditional AI/ML
2. Second Wave: Generative AI (ChatGPT)
3. **Third Wave: Agentic AI** ‚Üê Hum yahan hain
