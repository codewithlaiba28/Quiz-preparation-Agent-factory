## **The Technical Engine — Agent Factory Book Se**

Book ke mutabiq, **Claude Code ek Technical Engine hai** jo **General Agent se Custom Agents aur Skills generate karne me help karta hai**.   

---

### **1. Concept — Why a General Agent can build a Custom Agent**

* Pehle AI sirf **text generators** the (GPT jaise)
* Ab AI **action engines** ban gaye hain
* Yani ye **na sirf soch sakte hain aur text likh sakte hain**, balkay **real-world instructions aur workflows execute bhi kar sakte hain**   

---

### **2. The Logic**

* Claude Code **General Agent ka core engine** hai
* Ye **goal-driven** aur **spec-driven development** ka use karta hai
* Human goal/spec ko input me leta hai aur **necessary code aur SKILL.md instructions generate karta hai**   

---

### **3. The Loop — Read → Think → Code → Execute**

Claude Code ka workflow ek **continuous loop** me kaam karta hai:

1. **Read** — Spec aur documentation (SDKs/APIs) ko analyze karta hai
2. **Think** — Decide karta hai kaun sa tools, workflows aur logic use karna hai
3. **Code** — Custom SKILL.md ya Python / SDK code generate karta hai
4. **Execute** — Workflow ko run karta hai aur result check karta hai

* Ye loop **self-correcting aur iterative** hai, complex tasks me bhi kaam karta hai   

---

### **4. The Outcome**

* **Code = Universal Interface**
* Code sirf programming tool nahi; ye **human goal → actionable workflow** ka bridge hai
* Is wajah se **General Agent**:

  * SKILL.md likh sakta hai (instructions/logic)
  * Python / SDK code generate kar sakta hai
* Aur ye **har business domain me kaam karta hai**:

  * Data analytics
  * Document processing
  * Finance
  * HR workflows
  * Customer support   

---

### **Stepwise Flow (Book Style)**

1. **Human** → Goal / Spec define karta hai
2. **Claude Code** → Read, Think, Code, Execute loop me kaam karta hai
3. **Output** → Custom Skill (SKILL.md) ya Custom Agent (SDK code) ready
4. **Universal Interface (Code)** → Sab connect karta hai aur deployable banata hai

---

### **Exam-Ready One-Line Summary (Roman Urdu)**

> **Claude Code Technical Engine ke taur par kaam karta hai, jo Read-Think-Code-Execute loop me chal kar General Agent ko har business task ke liye Custom Skills aur Custom Agents generate karne ke qabil banata hai, aur Code iska universal interface hai.**
