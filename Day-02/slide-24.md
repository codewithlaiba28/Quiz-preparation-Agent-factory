## **Code is the Universal Interface**

### **Business Questions ka Code se Jawab kyu milta hai?**

---

## **Paradigm Shift**

* Pehle: Code sirf software banane ke liye likha jata tha
* Ab: Code **reality ko interrogate (pucha aur analyze) karne** ka tool ban gaya

Yani **code sirf programming nahi, problem-solving ka medium hai**

---

## **Example Scenario**

> Question: “Q3 me sales kyun drop hui?”

### **Coding Agent ka approach**

* Sirf code syntax aur comments pe focus karega
* Ye question **code ke context me samjhega**, problem ka answer nahi dega

### **General Agent ka approach**

1. **SQL query likhta hai** — sales data fetch karne ke liye
2. **Python script likhta hai** — trend visualize karne ke liye
3. **Chart analyze karta hai**
4. **Answer generate karta hai**:

> “Sales drop hui kyunki Enterprise sector me 40% churn hua”

---

## **Core Idea (Exam Ready)**

* General Agent **business analyst ki tarah kaam karta hai**
* Human question ko **code execution me translate karta hai**
* Faktaal, reliable answer deta hai, sirf guess nahi

---

## **One-Line Summary (Roman Urdu)**

> General Agents code ko **universal interface** ke taur par use karte hain jahan wo human questions ko factual, data-driven answers me convert karte hain.
