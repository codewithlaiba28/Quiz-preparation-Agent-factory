## **General Agents (Claude Code, Goose)**

### **The Power of Generalization**

Is ka matlab hai ke ye agents **har tarah ke kaam** handle kar sakte hain, sirf aik specific kaam tak limited nahi hote.

### **What it is**

Ye aik **autonomous agent** hota hai jo **terminal ya developer ke environment ke andar rehta hai**.
Isay bar-bar instructions dene ki zarurat nahi hoti.

---

## **Key Features**

### **Zero-Shot Planning**

Aap sirf **goal batate ho**,
agent **khud decide karta hai**:

* kya steps lene hain
* kis order me kaam karna hai

Yani planning bhi agent khud karta hai.

---

### **Deep Integration**

Ye agent:

* local files access kar sakta hai
* git history dekh sakta hai
* command line se direct kaam karta hai

Is liye ye **real developer jaisa kaam** karta hai.

---

### **Enhanced by MCP (Model Context Protocol)**

MCP ki wajah se agent:

* databases
* APIs
* external systems

ko **turant plug in** kar sakta hai, bina complex setup ke.

---

### **Enhanced by Agent Skills**

Agent Skills ka matlab:

* reusable modules
* organized folders
* instructions + scripts + resources

Is se agent **aur powerful aur flexible** ban jata hai.

---

## **Best For (Kis kaam ke liye best hai)**

### **Complex Debugging & Coding**

Jab code mushkil ho
ya bug samajhna ho
to ye agent **bohat effective** hota hai.

wahan General Agent best hota hai.

## **Thinking Loop Optimization**

Ye agent **loop me kaam karta hai**:

1. Read (samajhta hai)
2. Think (sochta hai)
3. Act (action leta hai)
4. Correct (galti sudhaarta hai)

Isi liye isay **true Agent** kaha jata hai, sirf chatbot nahi.
