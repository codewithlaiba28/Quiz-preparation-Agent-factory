## **Infinite Extensibility via MCP and Agent Skills**

### **Breaking the “Coder” Stereotype**

---

## **The Enabler**

* **MCP (Model Context Protocol)** + **Agent Skills**
  Ye do cheezen **General Agent ko har kaam ke liye extendable** banati hain

---

## **How it Expands the Role**

Agar hum agent me **MCP aur Skill plug in** karte hain, to agent ka role **poori tarah change ho jata hai**:

1. **Slack MCP + Communication Skill → Communications Manager**

   * Messages bhejna, teams me interact karna

2. **Salesforce MCP + Sales Skill → RevOps Specialist**

   * Sales workflows manage karna, CRM tasks

3. **Xero/QuickBooks MCP + Finance Skill → Financial Auditor**

   * Accounts check karna, audits, invoices

---

## **Why Coding Agent Cannot Do This**

* Traditional **Coding Agent**: strictly coding tasks tak limited
* Nahi bhej sakta messages, nahi audit kar sakta finances
* Scope limited hota hai

### **General Agent Advantage**

* General Agent ye sab kar sakta hai
* **Bas MCP (hands) aur Agent Skills (procedures) provide karo**

---

## **Core Idea (Exam Focus)**

> General Agents ko hum **“General”** is liye kehte hain:
>
> * Agar Finance MCP + Skill plug in ki → Finance Agent ban jata hai
> * Agar Sales MCP + Skill plug in ki → Sales Agent ban jata hai
> * Code sirf **glue / medium** hai jo sab cheez ko connect karta hai

---

## **One-Line Exam Ready Summary (Roman Urdu)**

> MCP aur Agent Skills ki wajah se General Agents ka role **infinite extendable** hai — ek hi agent ko communication, finance, ya sales expert banaya ja sakta hai, aur code sirf **unifying tool** hai.
