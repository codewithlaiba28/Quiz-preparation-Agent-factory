## **The Engine of Automated Automation — Agent Factory Book Se**

Book ke mutabiq ye concept batata hai ke **Claude Code ek engine hai jo Spec-Driven Development ke zariye Custom Skills aur Custom Agents create karta hai**.   

---

### **1. General Agent = Engine**

* **Claude Code** ko hum engine ke taur par samajhte hain
* Ye **spec-driven instructions** ko input ke taur par leta hai
* Aur un instructions se **production-ready code aur workflows** generate karta hai
* Ye engine **fully autonomous** hai, humans sirf goal/spec define karte hain   

---

### **2. Spec-Driven Development**

* Tum **Markdown Spec** me goal define karte ho
* Engine (Claude Code) spec ko **analyze + interpret** karta hai
* Necessary **tools, APIs, aur workflows** identify karta hai
* Ye approach **manual coding se fast aur repeatable** hai

> Spec-driven ka matlab:
> Spec = “What to do”
> Engine = “How to do it” automatically decide karta hai   

---

### **3. Output: Custom Skills & Custom Agents**

Claude Code **do cheezen generate karta hai**:

1. **Custom SKILL.md + Supporting Code**

   * Ye skill ek modular workflow hai
   * Agent ke liye “instructions + logic” ka set hai
   * Reusable aur sharable   

2. **Custom Agents Code**

   * SDK ke zariye (OpenAI Agents SDK, Claude Agents SDK) fully deployable agents generate hote hain
   * Ye agent ready hai real-world tasks execute karne ke liye   

---

### **4. Code = Universal Interface**

* Jo bhi generate hota hai (Custom Skill ya Custom Agent), **Code hi unka medium hai**
* Code **sirf software likhne ka tool nahi**, balkay **knowledge + workflow execution ka universal interface** hai
* Is interface ki wajah se General Agents har domain me kaam kar sakte hain   

---

### **Stepwise Flow (Book Style)**

1. **Human** → Goal / Spec define karta hai
2. **Claude Code (General Agent)** → Spec analyze + tools & APIs identify
3. **Manufacturing** → Custom Skills.md aur Supporting Code generate
4. **Output** → Deployable Custom Agent (via SDK) ready
5. **Code** = Interface jo sab connect karta hai

---

### **Exam-Ready One-Line Summary (Roman Urdu)**

> **Claude Code engine ke taur par kaam karta hai, jo Spec-Driven Development use karke Custom Skills aur Custom Agents generate karta hai, aur Code unka universal interface hai jo deployment aur execution enable karta hai.**
