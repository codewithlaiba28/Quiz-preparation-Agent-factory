### **1. Task Type**

**General Agent**

* Jab task **naya (novel)** ho
* Problem solving chahiye
* Clear steps pehle se defined na hon

---

### **2. End User**

**General Agent**

* Jab user **developer ya technical staff** ho
* Jo AI ke output ko samajh kar adjust kar sakta ho

**Custom Agent**

* Jab user **non-technical ya customer** ho
* Jahan simple, safe aur predictable behavior chahiye

---

### **3. Error Tolerance**

**General Agent**

* **High error tolerance**
* Human loop me hota hai (human check karta hai)
* Thodi creativity aur mistakes acceptable hoti hain

**Custom Agent**

* **Low error tolerance**
* Galti ki gunjaish nahi hoti
* Output reliable aur consistent hona chahiye

---

### **4. Cost Sensitivity**

**General Agent**

* **Cost per task zyada** hoti hai
* Lekin task ki value bhi high hoti hai
* Quantity kam, impact zyada

**Custom Agent**

* **Cost sensitivity high** hoti hai
* Thousands of tasks process karne hote hain
* Volume optimization zaroori hota hai

---

### **5. Implementation Time**

**General Agent**

* **Instant use**
* Install karo aur run karo
* Koi heavy design ya planning nahi

**Custom Agent**

* **Weeks lag sakte hain**
* Workflow design
* Guardrails
* Testing & deployment
