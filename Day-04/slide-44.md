**Agent Skills: Official Open Standard** (19 December 2025)

**The Announcement:**
Anthropic ne Agent Skills ko **open standard** ke taur par release kiya hai. Yeh ek specification aur reference SDK hai jo kisi bhi agent ko skills integrate karne ki ability deti hai.

**Kya hai yeh?**
- Bilkul MCP (Model Context Protocol) ki tarah open standard
- Universal adoption ke liye banaya gaya
- Website: agentskills.io aur GitHub repository available

**Official Adopters (Do Categories):**

**1. Native Support:**

**Enterprise Partners:**
Bari companies already use kar rahi hain: Atlassian, Canva, Figma, Notion, Ramp, Sentry, Cloudflare

**Book ke Mutabiq:**
Agent Factory book emphasize karti hai ke **agent-native development** ka matlab hai agents ko modular, reusable skills dena. Yeh open standard exactly wohi karta hai - ab koi bhi developer skills bana sakta hai jo kisi bhi agent platform par kaam karein.

**Tashbeeh:**
Jaise USB ne sab devices ke liye ek standard banaya, waise hi Agent Skills har agent ke liye ek common format bana raha hai!
